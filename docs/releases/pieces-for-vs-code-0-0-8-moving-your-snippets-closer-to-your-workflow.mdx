# Pieces for VS Code 0.0.8 - Moving your snippets closer to your workflow
> Mon Mar 27 2023 14:09:06 GMT+0000 (Coordinated Universal Time)
<p>We’re excited to launch a major release for Pieces for VS Code today.</p><p>One of our product principles has always been to <strong>keep developers in their flow</strong>, and today’s release is a big step forward where developers can see their saved snippets directly inside VS Code in a new Tree View.</p><p>Check out all the updates below, and look for the little sneak peak of what’s in our next big VS Code release - auto-complete!</p><h2 id="echtq">Visual Studio Code Extension</h2><p><em>Get the Pieces VSCode Extension <a href="https://marketplace.visualstudio.com/items?itemName=MeshIntelligentTechnologiesInc.pieces-vscode" target="_blank" rel="noopener noreferrer">here</a>.</em></p><h3 id="1bkkt">Saving Snippets</h3><ul><li>In addition to right-clicking or using keyboard shortcuts to save a snippet, users can now run the “Save to Pieces” command prompt.</li></ul><h3 id="dmbf9">Viewing Snippets inside VS Code</h3><ul><li>View your saved snippets inside VS Code via a Tree View in the sidebar! (You can always toggle to Pieces app for a more robust, dedicated snippet viewing and managing experience.)</li><li>To preview the contents of a snippet, simply hover your cursor over the name of the snippet.</li><li>Snippets in the VS Code Tree View are auto-organized by language</li><li>Refresh the list of snippets in the Tree View by clicking on the “Refresh” button in the upper right-hand corner of the Tree View.</li></ul><h3 id="fodth">Re-Using Snippets</h3><ul><li>To insert a saved snippet, click on the desired snippet in the Tree View and it will be inserted into your cursor location in your editor</li></ul><h3 id="2arll">Coming Soon: Auto-Complete Code in VS Code</h3><ul><li>Pieces will suggest code completions right in VS Code from your saved snippets in Pieces. Not from public repos like other auto-complete tools - Pieces auto-completes from YOUR private Pieces repo. Your code never leaves your machine.</li></ul><h2 id="efjia">Get Started with Pieces!</h2><p>As a reminder, make sure you have Pieces OS installed and running to use Pieces for VS Code. Pieces OS is the background service that enables Pieces to run locally.</p><p>Download Pieces OS for Mac <a href="https://updates.pieces.app/announcements/code.pieces.app/install" target="_blank" rel="noopener noreferrer">here</a> or for Windows <a href="https://www.microsoft.com/en-us/p/pieces-os/9n905ct3fpll?rtc=1&amp;activetab=pivot%3Aoverviewtab" target="_blank" rel="noopener noreferrer">here</a>.</p>\n\n    <figure data-rt-type="vide
