# New! Pieces CLI for MacOS
> **Published by the Pieces Team** | May 1st, 2023

The brand-new Pieces Command Line Interface (CLI) is now available for install on Mac. We’re thrilled to be bringing Pieces into your terminal, where we can help you save and re-use your snippets quickly and efficiently.
With this initial release, the Pieces CLI has only the most basic functionalities, but more features are coming quickly down the pipeline, including support for Windows and Linux! Stay tuned for updates.

## Install
To install the Pieces CLI, run the following commands:
```brew tap pieces-app/pieces ```
```brew install pieces```

> IMPORTANT: If you’ve never used Pieces before, you’ll also need to install Pieces OS, the background service that enables Pieces to work locally and handles communications with any integrations, including the CLI.
