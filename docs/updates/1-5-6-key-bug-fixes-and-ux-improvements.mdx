<p>At Pieces, we strive to place as much emphasis on the little things as we do on the big things. That&#x27;s why we pay extra attention to detail to make sure you have the smoothest user experience possible. This release doesn&#x27;t include any fancy new features, but rather many small bug fixes and UX improvements users have noticed along their journey.
