# 1.5.0: Extract code from screenshots and more
> Mon Mar 27 2023 14:06:02 GMT+0000 (Coordinated Universal Time)
<p>A few major features in this release, the biggest being that you can now drag in screenshots of code, and Pieces will
    automatically convert it to copyable text. Read below for more new features and UI changes.</p><h2 id="9fihi">Pieces
    for Developers</h2><h3 id="4ftth">Adding Pieces</h3><ul>
    <li>[New Feature] Users can now add PNG code screenshots to Pieces via manual file attachment or drag and drop</li>
    <li>Screenshots will run through optical character recognition analysis (OCR) to enable users to use code from
        within a screenshot without retyping
    </li>
</ul><h3 id="8qan7">Finding Pieces</h3><p>[New Feature] Users can now add custom tags to a piece to help with search</p>
<p>Users will now see a helpful keyboard shortcut indicator in the search bar to help navigate search behavior: </p><ul>
    <li>Click “/” to focus the search</li>
    <li>Click “esc” to escape or re-focus search</li>
    <li>Click “return” to navigate through search results</li>
</ul><h3 id="s4cm">Navigating Pieces</h3><figure class="w-richtext-figure-type-image w-richtext-align-center"
