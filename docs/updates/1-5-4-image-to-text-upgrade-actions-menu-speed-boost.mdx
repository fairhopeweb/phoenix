# 1.5.4: Image to Text upgrade, Actions menu, speed boost
> Mon Mar 27 2023 14:17:15 GMT+0000 (Coordinated Universal Time)
<p>We&#x27;ve got a jam-packed release today that includes a major OCR model upgrade for our popular text extraction from screenshots feature, major performance improvements and some bug fixes. Reach out to us on <a href="https://www.linkedin.com/company/getpieces" target="_blank" rel="noopener noreferrer">LinkedIn</a>, <a href="https://twitter.com/getpieces" target="_blank" rel="noopener noreferrer">Twitter</a>, <a href="https://code.pieces.app/" target="_blank" rel="noopener noreferrer">Live Chat</a> or support@pieces.app with questions or feedback.</p><h2 id="cjcdp">Pieces for Developers</h2><p><em>Pieces for Developers is the core user-facing Pieces application.
