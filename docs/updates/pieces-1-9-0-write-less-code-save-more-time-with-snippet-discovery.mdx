# Pieces 1.9.0: Write less code, save more time with Snippet Discovery
> Mon Mar 27 2023 15:59:21 GMT+0000 (Coordinated Universal Time)
<p>Introducing Snippet Discovery— the fastest, easiest way to discover and save useful snippets from your own code. Snippet Discovery scans a given file or directory and finds the code snippets that you use frequently, enabling you to save them to Pieces. Even better, your saved snippets are upgraded to include smart tags, descriptions and more.</p><figure class="w-richtext-figure-type-video w-richtext-align-center" style="padding-bottom: 33.9%;" data-rt-type="video" data-rt-align="center" data-rt-max-width="" data-rt-max-height="33.9%" data-rt-dimensions="200:113" data-page-url="https://www.youtube.com/watch?v=pFvMxo___Rw"><div><iframe width="200" height="113" src="https://www.youtube.com/embed/pFvMxo___Rw?feature=oembed" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen title="Introducing Snippet Discovery - Extract Code Snippets from your Files"></iframe></div></figure><p>Discovering snippets in your existing code has huge benefits, including:</p><ul><li>Standardizing your code within and across projects</li><li>Less time spent re-writing the same code over and over</li><li>Quickly and easily sharing standard snippets among your team</li></ul><h2 id="9kkqs">How to Use Snippet Discovery</h2><p>To activate Snippet Discovery:</p><ol type="1"><li>Open the “Add a Piece” submenu</li></ol><figure class="w-richtext-figure-type-image w-richtext-align-center" data-rt-type="image" data-rt-align="center"><div><img alt="The Add a Piece menu in the Pieces Desktop App." src="https://uploads-ssl.webflow.com/6143afec68f555387049efb3/632a3ad146a8ea6fbbd4b05d_ScreenShot2022-09-19at4_54_32PM_f2eaef9e6a443ea68ac51822826801b4_800.png" /></div></figure><p>2.
