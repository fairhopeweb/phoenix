# Pieces Release 1.4 - VS Code and UltraEdit integrations!
> Mon Mar 27 2023 13:42:22 GMT+0000 (Coordinated Universal Time)
<h2 id="8akfk">Pieces OS</h2><p><em>For newer users, Pieces OS is the background service that enables Pieces to work
    locally and handles communications with any integrations you install, among many other tasks.</em></p><ul>
    <li>Support for automatically launching Pieces OS on machine boot-up</li>
    <li>Pieces database file moved to a non-temp location</li>
    <li>New redirect pages for successful sign-in &amp; sign-out</li>
</ul><h2 id="5hcld">Pieces for Developers</h2><p><em>For newer users, Pieces for Developers is the core user-facing
    Pieces application.</em></p><h3 id="7rf3g">Saving Pieces</h3><ul>
    <li>Users now receive an in-app message when creating a piece to reclassify the code if the classification
        prediction confidence is low
    </li>
</ul><h3 id="daru1">Viewing and Managing Your Pieces</h3><ul>
    <li>Users can now zoom in/out with [CMD +] or [CMD -]</li>
    <li>Users can now right-click anywhere to open the quick menu</li>
    <li>Users can now triple-click to select all code in the preview</li>
    <li>Renaming repopulates previous name if user edits the name and closes without saving</li>
    <li>Improved support for long piece names</li>
    <li>Improved navigation between drawers - click [esc] to navigate back to Quick Menu from submenus (rename,
        description, reclassify, delete)
    </li>
    <li>Users can now see where a piece was saved from (its &quot;origin&quot;) in the Quick Menu</li>
    <li>Users can now click to the left and right of menus to exit/collapse</li>
</ul><h3 id="fao8r">Search</h3><ul>
    <li>Users can now search by description</li>
    <li>Bug fixes to search string highlighting</li>
</ul><h2 id="d9e1h">Pieces Integrations</h2><p><em>For newer users, you can <a
    href="http://code.pieces.app/integrations">install integrations here</a> that enable in-app saving and retrieving of
    Pieces inside your favorite code editors and Chrome.</em></p><h3 id="b7cn1">UltraEdit</h3><ul>
    <li>Pieces for UltraEdit now in beta in partnership with our friends at UltraEdit</li>
    <li>Save code directly from within UltraEdit</li>
    <li>View your pieces directly within UltraEdit</li>
    <li>Much more coming soon</li>
</ul><h3 id="5qdrh">VS Code</h3><ul>
    <li>Released Pieces for VS Code</li>
    <li>VS Code users can now save pieces from within VS Code using the right-click context menu or the [shift + cmd +
        v] keyboard shortcut (ctrl on windows)
    </li>
</ul><h3 id="9n2nq">Pieces API</h3><ul>
    <li>Released a Connector API that will significantly improve developer experience when building integrations on
        Pieces OS
    </li>
    <li>Infrastructure completed for upcoming Atom integration</li>
    <li>Initial groundwork created for &#x27;&lt;application&gt;/suggestion&#x27; API endpoint</li>
    <li>Initial groundwork created for &#x27;&lt;application&gt;/reaction&#x27; API endpoint</li>
</ul>\n\
