## 3 Steps to Get Up and Running with Pieces for VS Code

### 1. Save Your First Snippet

<Image src={"/assets/graphics/vscode/onboarding_save_vscode.png"} />

Highlight, right-click and choose "Save to Pieces", or use the keyboard shortcuts.

- macOS: ```shift + ⌘ + v```
- Windows/Linux: ```shift + ctrl + v```

### 2. Find & use your snippets

<Image src={"/assets/graphics/vscode/onboarding_find_vscode.png"} />

Access your snippets by navigating to the Pieces for VS Code List View by clicking the Pieces for Developers logo on the left-hand Action Toolbar.

Also, you can search for and view your snippets using the VS Code Command Palette keyboard shortcut:

- macOS: ```⌘ + k```
- Windows/Linux: ```ctrl + k```

### 3. Use with Pieces Desktop App

<Image src={"/assets/graphics/vscode/onboarding_use_vscode.png"} />

Our VS Code Extension in combination with our <a target="_blank" href="https://pieces.app">Flagship Desktop App</a> provides users a seamless flow of developer materials in a standalone app.

- Get back in flow with our Workflow Activity View
- Save time with In-Project Snippet Discovery
- Enjoy real-time and scope-relevant suggestions
- Extract and use code and text from screenshots
